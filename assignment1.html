Q. Define web browser, web page , website and web server ? -Web browser - A browser or web browser is a program used to view page on the internet and navigate the world wide web . Web page - World Wide Web information is presented on the web pages , which you download to the computer using a web browser such as internet explorer . Website - A website is a collection of web pages associated with a particular preson , business , government , school and organization. Website are stored on a web server , a special computer that makes web pages available for people to browser. Web server - Web servers are computers whose job is to respond to a browsers request for a web page and deliver it through the internet . Pages hosted on a webserver can be displayed to anyone all over the world .
